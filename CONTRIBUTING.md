# Contributing

Thanks for your interest in Vartovii.

This repository is the public-facing surface of the product. Contributions here are mainly useful for improving:

- documentation clarity
- public architecture explanations
- broken links or stale references
- issue reporting and public-facing assets

## How To Contribute

1. Open an issue describing the improvement or problem.
2. Keep proposals concrete and scoped.
3. If submitting a pull request, prefer small, reviewable changes.
4. Do not include secrets, credentials, or internal environment details.

## Good Contributions

- documentation fixes
- wording improvements
- typo and formatting cleanup
- link validation and surface consistency
- public roadmap clarity

## Not Appropriate Here

This repository is not the place for:

- reporting security vulnerabilities publicly
- requesting access to private infrastructure
- sharing internal credentials or deployment artifacts
- reverse-engineering private services

For security issues, use the process in [`SECURITY.md`](SECURITY.md).
